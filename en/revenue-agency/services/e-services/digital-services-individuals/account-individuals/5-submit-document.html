The "Submit documents" service allows you to send documents electronically to the CRA. You can only use this service when:

  1. You would like to send relevant documents that are associated to a specific program through the CRA’s online services; or
  2. We have communicated with you requesting documents, and have invited you to use the “Submit documents” service, one of the CRA’s online services.

You can use this service in the following cases:

  • you do not have a case or reference number but know the applicable topic for your document submission and you are registered for the CRA’s online services; or
  • the CRA provided you with a case or reference number.

How to submit a document

If you have a case or reference number:

Step 1- Case or reference number

  1. Enter the case or reference number in the space provided. This number is either given to you by mail, by phone, in person by a CRA official, or on the confirmation screen of a previous document submission.
  2. Select “Next”.
  3. Select the tax year associated with the document you are submitting and select “Next”.

Step 2 – Upload files you want to send

  1. Make sure the file format you want to send is acceptable. The file must be in one of the following formats in order to be uploaded: .pdf, .doc, .docx, .xls, .xlsx, .rtf, .txt, .jpg, .jpeg, .tiff, .tif, or .xps.
  2. Enter a description of the file in the space provided. If more than one file is being uploaded, each must have a unique name.
  3. Use the “Browse…” button on the right to select from your computer the file to be uploaded.
  4. Select “Add file”.

Note

For compliance (audit) programs only - In addition to the acceptable file formats listed above, the CRA also accepts all safe file formats including text, document, spreadsheet, database, image, graphic, video, audio, accounting, financial, and tax files. Submitted files may be zipped but must not be password encrypted, as this will prevent the CRA’s system from scanning for viruses. Files that can’t be scanned for viruses will be securely deleted.

Step 3 – Review information before sending

  1. If you want to delete a file from this process, select it in the column on the left, then select “Delete selected file(s)”.
  2. If you want to upload another file, select “Add another file”.
  3. Select “Submit file(s)” once you have included all the files you want to send.

The total size of all files you send cannot exceed 150MB. A limit of ten files per submission may apply.

If you do not have a case or reference number:

Step 1 – Case or reference number

  1. Select “I do not have a case or reference number”.
  2. Select the topic that applies to your document submission.
  3. Select “Next”

Step 2 – Upload files you want to send

  1. Make sure the file format you want to send is acceptable. The file must be in one of the following formats in order to be uploaded: .pdf, .doc, .docx, .xls, .xlsx, .rtf, .txt, .jpg, .jpeg, .tiff, .tif, or .xps.
  2. Use the “Browse…” button on the right to select from your computer the file to be uploaded.
  3. Enter a description of the file in the space provided. If more than one file is being uploaded, each must have a unique name.
  4. Select “Upload file”.

Step 3- Review information before sending

  1. If you want to delete a file from this process, select it in the column on the left, then select "Delete file(s)".
  2. If you want to upload another file, select "Add another file".
  3. Select "Submit" once you have included all the files you want to send.

The total size of all files you send cannot exceed 150MB. A limit of ten files per session may apply.

Receiving the confirmation

  1. A confirmation screen will appear which includes a confirmation number and a reference number.
  2. Keep a record of your confirmation number and reference number. You can use this reference number to submit additional documents related to the original submission.
  3. You can print this page by using your browser's print button, or by selecting the "Print" button.

All files you send will be scanned for viruses. We will identify any unacceptable files and will contact you with instructions for sending the files again using a different method.
